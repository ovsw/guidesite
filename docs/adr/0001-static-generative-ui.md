# Use Static Generative UI

GuideSite uses Static Generative UI instead of arbitrary AI-generated UI. The front end owns the approved components, layouts, styles, interactions, and accessibility constraints; AI chooses and fills those components during each GuideSite Turn.

This gives visitors a personalized Session Timeline without letting AI invent markup, layouts, or interaction patterns. The trade-off is less open-ended generation, but much higher control, quality, brand consistency, accessibility, and safety.
